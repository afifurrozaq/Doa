# <img src="public/icon/doa.png" width="54"> Doa App

Simple web application that displays a collection of **daily Islamic prayers (Doa)** including Arabic text, Latin transliteration, and Indonesian translations.

This project was built as a simple frontend project to practice working with **structured data and UI rendering using React**.

## ✨ Features

- 📖 Collection of daily Islamic prayers
- 📖 Al-Qur'an
- 🕌 Prayer Notifications with Adzan
- 🕌 Arabic text display
- 🔤 Latin transliteration
- 🌍 Indonesian translation
- 📱 Responsive layout
- ⚡ Fast loading static web app

## ⚙️ Installation

Clone this repository:

```bash
git clone https://github.com/afifurrozaq/Doa.git
npm install
set .env VITE_GOOGLE_SHEET_ID=
npm run dev
```
